# DFT — Document & File Tools Platform

This repository contains the product and technical blueprint for a high-traffic document tools platform focused on PDF and office conversion workflows.

## Initial Tool Suite

- PDF Compressor
- PDF Merge
- PDF Split
- PDF to Word
- Word to PDF
- Image to PDF
- Online PDF Form Filler

See detailed strategy and implementation docs in [`docs/product-strategy.md`](docs/product-strategy.md) and [`docs/implementation-plan.md`](docs/implementation-plan.md).
